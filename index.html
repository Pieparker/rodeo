For https://answers.netlify.com/t/changes-to-redirects-with-query-string-parameters-are-coming/23436.

  • /urldummy id=:id /urldummy 301!
    https://redirect-dummy-query-string.netlify.rodeo/urldummy?id=1 will break (redirect loop).

  • /urldemo id=:id /urldemo?no-query-string 301!
    Try navigating to https://redirect-dummy-query-string.netlify.rodeo/urldemo?id=1, which will succeed and only use the query string in the to rule.

  • /urltest id=:id /urldemo?no-query-string
    Try navigating to https://redirect-dummy-query-string.netlify.rodeo/urldemo?id=1, which will succeed without the force (!) as /urltest doesn't exist.
